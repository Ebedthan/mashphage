# mashphage: Rapid whole-genome clustering of Actinobacteriophages


## Warning
mashphage is actually under regular updates and development. It should be therefore used with caution. Stable release will be available as soon as possible.

## Introduction

Actinobacteriophage are a diverse phylum of phages. It is composed of phages infecting bacterias economically important, useful in agriculture and of bacterial pathogens.

It is therefore an intensive studied phylum in the aspect of antimicrobial resistance. With comparative genomics, phages are sutdied using advance tools to analyze their genomes. Such analyses includes whole-genome clustering.

Here we present a faster and reliable method for actinophage whole-genome clustering: mashphage.
